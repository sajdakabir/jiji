How it stays out of the way

Small, local, and quiet.

🐱 Just a cat in the bar

No Dock icon and no window to manage — only a small cat.fill glyph up top. Click for the popover, click away to dismiss.

🔒 Stays on your Mac

Sign in once through an in-app claude.ai login window. The session cookie stays local. No servers, no API key, no telemetry.

🔄 Quietly up to date

Refreshes about once a minute by reading your claude.ai usage page. The percent in the menu bar is always current.

📖 Free & open source

The whole thing is on GitHub. Read it, build it, or send a patch — nothing hidden.

Install

Up and running in a minute.

  1. Download the latest .dmg
    Grab it from the releases page and open the disk image.
  2. Drag Jiji to Applications
    Move Jiji.app into your Applications folder, as usual.
  3. First launch: right-click → Open
    Jiji is ad-hoc signed (not notarized), so the very first time, right-click the app and choose Open, then confirm. After that it opens normally.
  4. Sign in to claude.ai once
    An in-app login window appears. Sign in, and the cat starts showing your usage. Click the menu bar 🐱 any time.
⚠️

Why the right-click? Jiji isn't notarized by Apple, so Gatekeeper asks once. Right-click → Open tells macOS you trust it. You only do this on the first launch.
